Pattern 1: Newspaper Layout with grid-template-areas

💡 What Learn: grid-template-areas, named grid regions
Understanding grid-template-areas:

This is visual way to define grid layout. You give regions names (like "header", "main", "sidebar") and show ASCII art of layout. CSS understand instantly! 

grid-template-areas:
    "header header"     /* header span 2 columns */
    "main sidebar"      /* main left, sidebar right */
    "footer footer";    /* footer span 2 columns */

Then on each element: grid-area: header, grid-area: main, etc.

Pattern 2: Masonry Grid with auto-fit + minmax()

💡 What Learn: auto-fit, minmax(), grid-auto-rows, fr unit
Key concepts:
  • fr unit = "flex row" = fraction of available space (like flex: 1)
  • minmax(200px, 1fr) = Minimum 200px, maximum 1 fraction of space
  • auto-fit = Automatically fit as many column as possible in available width
  • grid-auto-rows: minmax(200px, auto) = Rows auto-create if need, min 200px

Result: Masonry/gallery grid that responsive without media query!

🎓 Important Grid Concepts

What is fr unit?

fr = "fraction" of available space. Like flex: 1 in Flexbox. If grid-template-columns: 1fr 1fr 1fr (3 equal column) and width 900px, each column 300px.

px vs % vs fr

  • px = Fixed pixel size (100px always 100px)
  • % = Percentage of parent (50% = half of parent)
  • fr = Fraction of remaining space (flexible like flex)

What does minmax() do?

minmax(200px, 1fr) mean: minimum 200px, maximum 1 fraction of space. Useful for: "I want column at least 200px but can grow if space available".

auto-fill vs auto-fit

  • auto-fill = Create many track as fit, even empty one
  • auto-fit = Create track only if have content, empty collapse

grid-template-areas ASCII art

Show layout as ASCII art. Each name represent region. Must be rectangular, no weird shape. Visual layout immediately clear!

🛡️ Defense Questions

Q: grid-template-areas bhanne ke ho? Kasari use garchau? (What is grid-template-areas? How do you use it?)

A: grid-template-areas is visual way to define grid layout using ASCII art. You create named region (like "header", "main", "sidebar") and draw how they arrange. Then on element use grid-area: header to place element in that region. Very readable! 

grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }
Q: fr unit ko matlab ke ho? Ka difference px/% sanga? (What does fr unit mean? Difference with px/%?)

A: fr = "fraction" of available space. If grid-template-columns: 1fr 1fr 1fr, three equal column each taking 1 fraction. px = fixed size, % = percentage, fr = flexible remaining space (like flex: 1).

Q: minmax() function kina use garchau? (Why use minmax() function?)

A: minmax(200px, 1fr) set minimum and maximum size. Column must be at least 200px, but can grow up to 1 fraction of space. Perfect for: "I want card 200px min but responsive on big screen".

Q: auto-fill vs auto-fit ko difference? (What difference between auto-fill and auto-fit?)

A: auto-fill create many track as fit (maybe empty). auto-fit create only track with content, empty collapse. In 80% case use auto-fit!
