📚 Summary
Bootstrap 5 Data Attributes:
  • data-bs-toggle="collapse" → Collapsible element
  • data-bs-toggle="modal" → Show modal
  • data-bs-toggle="tab" → Tab switching
  • data-bs-target="#id" → Target element ID
  • data-bs-dismiss="modal" → Close modal

Note: These attribute work with Bootstrap JavaScript library. Bootstrap CDN include JavaScript file automatically.

In React: For simple CSS (table, badge, button) can use data-bs directly. For interactive (modal, accordion open/close), better use React state + useState hook instead of data-bs attribute. Better control, more React way.
