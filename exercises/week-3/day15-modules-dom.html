📝 How This Works (Module + Debounce)

ES6 Modules: This page imports helper functions from day15-helpers.js using type="module" script tag. The helpers include formatCurrency, truncateText, and debounce functions.

Debounce: The search box uses debounce to prevent excessive function calls. It only fires the search handler 300ms AFTER you stop typing. Without debounce, search would fire on every keystroke (performance issue!). This is key concept in modern web development.

// Debounce prevents "too many API calls" const debouncedSearch = debounce((query) => { filterProducts(query); }, 300); searchInput.addEventListener('input', debouncedSearch);

Defensive Note (for internship): Debounce bhanne ko = function ko execution delay garne. User type every keystroke, 300ms pachi request marey. Without debounce: 100 keystroke = 100 API call! With debounce: 100 keystroke = 1 API call. Network bandwidth ra server load save hunchha!
