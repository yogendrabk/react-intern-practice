How work:
  • display: flex on grid
  • flex-wrap: wrap — Items can go next line instead of shrink
  • gap: 1rem — Space between all items
  • flex: 1 1 250px on card — KEY PART!

Breaking down flex: 1 1 250px:

  • First 1 = flex-grow: 1 (grow if extra space)
  • Second 1 = flex-shrink: 1 (shrink if space tight)
  • 250px = flex-basis: 250px (natural size before grow/shrink)

Result: Each card try be 250px, if space available grow equal, if tight wrap to next line

🛡️ Defense Questions & Answers

Q: Flexbox bhanne ke ho? (What is Flexbox?)

A: Flexbox is CSS layout system for one-dimensional layout (row OR column, not both). Perfect for navigation, centering, spacing items. You have flex-container (parent with display: flex) and flex-items (children). Container controlling how items distributed, spaced, aligned.

Q: justify-content ra align-items ko difference ke ho? (What difference between justify-content and align-items?)

A: Both for alignment but different direction. If flex-direction: row (horizontal):

  • justify-content = Align horizontally (main axis)
  • align-items = Align vertically (cross axis)

If flex-direction: column (vertical): reverse — justify-content vertical, align-items horizontal.

Q: flex: 1 1 250px matlab ke ho? ko-ko meaning? (What does flex: 1 1 250px mean?)

A: This shorthand for three property:

  • First 1 = flex-grow: 1 (If container have extra space, grow 1 part of it)
  • Second 1 = flex-shrink: 1 (If container space tight, shrink 1 part)
  • 250px = flex-basis: 250px (Natural size before grow/shrink)

Real example: If 4 card with flex: 1 1 250px in 1000px container: Each try 250px (total 1000px). Perfect! All equal size. In 600px container: Each 150px (shrink equally).

Q: flex-wrap: wrap kina use garchau? (Why use flex-wrap: wrap?)

A: By default flex-wrap: nowrap (all items in single line, squeeze if need). With flex-wrap: wrap, items go next line instead of squeeze. Perfect for responsive card grid. On big screen 4 cards in row. On small screen wrap to 2 cards, then 1 card.
