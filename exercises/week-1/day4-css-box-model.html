🎓 Box Model Explanation

CSS Box Model is fundamental concept in web design. Every element in HTML have box around it consisting of 4 layer:

1. MARGIN - Outside spacing. Space between this element and neighbor element. Internet is transparent.

2. BORDER - Line around element. Can have color, style (solid, dashed, etc), and thickness.

3. PADDING - Inside spacing. Space between content and border. Inside element.

4. CONTENT - Actual content (text, image, etc) of element. Innermost layer.

total width = content width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Important: By default, when you set width: 200px, it only set content width. Padding, border, margin add on TOP of this! This can confusing. Solution: Use box-sizing: border-box - make width include padding and border.

🛡️ Defense Questions & Answers

Q: CSS box model explain gara? (Explain CSS box model)

A: CSS Box Model consisting of 4 layer: MARGIN (outside spacing), BORDER (line around), PADDING (inside spacing), CONTENT (actual element). Think like book: MARGIN = space from table edge, BORDER = book cover, PADDING = margin inside book cover, CONTENT = actual text in book. Understanding box model important for layout and sizing element correctly.

Q: box-sizing: border-box kina use garchau? (Why do I use box-sizing: border-box?)

A: By default, when I set width: 200px, it only including content width. Padding and border add on top, so total width become more than 200px. This confusing! With box-sizing: border-box, width include padding and border. So width always exactly 200px. Easier to calculate and predict layout.

/* Without border-box */
width: 200px; padding: 10px; /* Total = 220px */

/* With border-box */
box-sizing: border-box;
width: 200px; padding: 10px; /* Total = 200px */
Q: Difference between margin and padding ke ho? (What is difference between margin and padding?)

A: MARGIN = outside space, creating distance between this element and neighbor element. Does not have background color (transparent). PADDING = inside space, creating distance between content and border. Have background color of element. Think like: margin pushing neighbor away, padding giving breathing room to content.

Q: Element ko width 300px set garyo, but screen ma 350px dekhiyo. Kina? (Why is width different?)

A: This happen because of box model! If me set width: 300px, padding: 20px, border: 5px, then total width = 300 + 20 + 20 + 5 + 5 = 350px. This called "content-box" (default). Solution: Add box-sizing: border-box so 300px include padding and border.
