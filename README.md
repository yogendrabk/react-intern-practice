## 🚀 Getting Started

### Installation

```bash
# Navigate to app folder
cd yogendra-react-app

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will run at `http://localhost:5173`

### Build for Production

```bash
npm run build
npm run preview
```
